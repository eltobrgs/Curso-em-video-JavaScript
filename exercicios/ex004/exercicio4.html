Este script solicita ao usuário que insira seu nome, idade e saldo bancário, e em seguida exibe várias informações com base nas entradas do usuário.

  1. window.prompt('mensagem'): Esta função exibe uma caixa de diálogo que solicita ao usuário que insira algum texto. A mensagem fornecida é exibida na caixa de diálogo.
  2. document.write('mensagem'): Esta função escreve uma string de texto em um documento HTML. Se o documento já foi processado, a função document.write() irá sobrescrever o conteúdo existente.
  3. nome.length: Esta propriedade retorna o número de caracteres na string nome.
  4. nome.toUpperCase(): Esta função retorna a string nome convertida em letras maiúsculas.
  5. nome.toLowerCase(): Esta função retorna a string nome convertida em letras minúsculas.
  6. nome.split(' '): Esta função divide a string nome em um array de substrings, usando um espaço como separador.
  7. nome.join(''): Esta função junta os elementos de um array em uma única string, sem nenhum caractere entre eles.
  8. nome.charAt(0): Esta função retorna o caractere na posição especificada (neste caso, 0) na string nome.
  9. nome.slice(0,3): Esta função retorna uma substring da string nome, começando na posição 0 e terminando antes da posição 3.
  10. nome.slice(-3): Esta função retorna uma substring da string nome, começando na terceira posição a partir do final da string e indo até o final da string.
  11. nome.slice(2): Esta função retorna uma substring da string nome, começando na posição 2 e indo até o final da string.
  12. Number.parseInt(idade): Esta função converte a string idade em um número inteiro.
  13. (banco*0.9).toFixed(2): Esta expressão primeiro multiplica o valor de banco por 0.9, e então usa a função toFixed(2) para formatar o resultado como uma string com duas casas decimais.
